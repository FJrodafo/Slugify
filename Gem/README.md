## Install it!

```shell
gem install fjrodafo-slugify
```

## Run it!

```ruby
# Load the gem
require "fjrodafo/slugify"
```

```ruby
Fjrodafo::Slugify.to_slug("Hello, World!")
# => "hello-world"
```
